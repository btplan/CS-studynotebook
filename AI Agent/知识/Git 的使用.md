| 场景       | 命令                     |
| -------- | ---------------------- |
| 看项目状态    | `git status`           |
| 添加修改     | `git add .`            |
| 保存版本     | `git commit -m "说明"`   |
| 查看历史     | `git log --oneline`    |
| 回到旧版本    | `git checkout xxx`     |
| 强制回滚     | `git reset --hard xxx` |
| 打里程碑     | `git tag v1.0`         |
| 上传GitHub | `git push`             |
